# Simrit Gyan - Home Tuition Platform

A modern Next.js website for Simrit Gyan, a registered home tuition company.

## Features

- Responsive design for all devices
- Modern UI with gradient themes
- Service showcase
- Contact form
- About section
- SEO optimized

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm start
```

## Customization

- Update contact information in `components/Contact.tsx` and `components/Footer.tsx`
- Modify colors in CSS files to match your brand
- Add more services in `components/Services.tsx`
- Update company details in `components/About.tsx`

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules
